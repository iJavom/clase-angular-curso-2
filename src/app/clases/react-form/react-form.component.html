React Form

React form forma parte de la libreria ReactiveFormsModule de angular y nos sirve para crear formularios de manera reactiva. Gracias a esta libreria no dependemos del ngModel y nos facilita los proceso de manejo de valores, validaciones e incluso podemos crear nuestras propias validaciones.




Es indispensable importar en el module a ReactiveFormsModule y en el html tenemos que trabajar dentro de un elemento form que tendra la directiva formgroup , adicional cada input debe ir identificado con formControlName
